# Архитектура проекта — компоненты

### Компоненты

1) База знаний

- Формат исходных материалов: Markdown/YAML
- Структура: теги, граф связей
- Версионирование: GitHub

2) Экосистема AI‑агентов

- Роли: Vova (музыка), Petrova (визуал), Hugo (парсинг), ИИ (метаагент), Dizzy (дизайн), Nay (графы), Cadric (CAD/3D), гости

3) Автоматизация

- Сбор данных (HuggingFace, сайты производителей)
- Обогащение LLM
- Генерация markdown‑страниц
- Публикации в соцсети/на сайт

4) Контент‑генерация

- YouTube, Telegram, сайт‑портфолио

- Персонажи (Агенты)
