# ComfyUI — обзор и быстрый старт

## TL;DR
- ComfyUI — конструктор workflow для Stable Diffusion; собираем пайплайны без кода.
- Подходит для быстрых экспериментов, повторяемых пресетов и демонстраций процессов.
- Стартовая связка: загрузить шаблон, поменять входные параметры, сохранить результат.

## Быстрый старт
1. Установите приложение из [репозитория ComfyUI](https://github.com/comfyanonymous/ComfyUI).  
2. Запустите UI и импортируйте базовый workflow (`Load` → `Examples`).  
3. Подключите чекпойнт и VAE, настройте резолюцию, seed и prompt.  
4. Запустите граф, сохраните пресет и экспортируйте изображения или видео.

## Частые ошибки
- **Конфликт версий узлов.** Проверяйте зависимости, обновляйте кастомные ноды.  
- **VRAM заканчивается.** Уменьшайте размер, используйте `fp16` или offload на CPU.  
- **Несохранённые пресеты.** Держите библиотеку workflow в git или облаке, чтобы делиться с командой.

## Как встроить в процесс
- Готовые пресеты подключаем к пайплайнам публикаций (Notion → GitHub → сайт).  
- Результаты прикладываем в портфолио и кейсы — удобно показывать пайплайн клиентам.  
- Для автоматизации используем экспорт графа и запуск через CLI.
