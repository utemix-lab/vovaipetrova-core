# Процесс: Обсудили → Разложили → Связали

### Принцип

Тема → Распределение по разделам → Взаимосвязанность

Коротко: «Обсудили → Разложили → Связали»

### Как применяем

1) Обсуждаем аспект и фиксируем 3–5 тезисов.

2) Раскладываем по разделам: Навигация (пользовательская/техническая), Think Tank, Услуги/Автоматизация, Портфолио, База знаний.

3) Ставим двусторонние ссылки и короткие аннотации «зачем перейти».

4) Обновляем «Индекс сайта».
