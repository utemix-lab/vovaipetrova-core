# Backlinks

## TL;DR

Backlinks (входящие ссылки) — это список всех страниц, которые ссылаются на текущую страницу. Генерируются автоматически и используются для навигации и понимания связей между документами.

## Что это

Backlinks:
- Показывают, какие страницы ссылаются на текущую
- Помогают находить связанные материалы
- Используются для построения графа знаний

## Генерация

Backlinks генерируются автоматически скриптом `scripts/generate-backlinks.mjs`:
- Анализирует все страницы проекта
- Извлекает ссылки из Markdown контента
- Строит обратный индекс: страница → список ссылающихся страниц

## Хранение

Backlinks хранятся в `prototype/data/backlinks.json`:

```json
{
  "page-slug": [
    {
      "slug": "referring-page-slug",
      "title": "Referring Page Title",
      "url": "docs/referring-page.md"
    }
  ]
}
```

## Отображение

Backlinks отображаются в сайдбаре страницы Explorer:
- Секция "Ссылки на эту запись"
- Список ссылающихся страниц с кликабельными ссылками
- Автоматически обновляется при каждом билде

## Использование

### В навигации

Backlinks помогают:
- Находить связанные материалы
- Понимать контекст использования термина
- Строить граф знаний проекта
