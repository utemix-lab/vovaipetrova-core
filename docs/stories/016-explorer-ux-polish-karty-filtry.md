# Explorer UX polish: карты, фильтры, breadcrumbs

TL;DR

- Улучшены карточки: summary обрезается до 120-140 символов с ellipsis.
- Добавлены фильтры по статусу (all, ready, review, draft) и тегам (clickable chips).
- Реализован toggle "Ready only" для показа только готовых страниц.
- Добавлены breadcrumbs на страницах эпизодов: Home / Stories / Title.
- Дальше — дальнейшее развитие UX и добавление новых возможностей навигации.

**Что произошло.** После создания Explorer потребовалось улучшить пользовательский опыт навигации. Улучшены карточки с обрезкой summary, добавлены фильтры по статусу и тегам, реализован toggle "Ready only", добавлены breadcrumbs на страницах. Это сделало Explorer более удобным для навигации по контенту.

**Зачем это делали.** Нужна была возможность быстро находить нужный контент и понимать структуру навигации. Фильтры позволяют сужать выборку по статусу и тегам, breadcrumbs показывают текущее местоположение, обрезка summary делает карточки более читаемыми.

**Что получилось.** Карточки получили обрезку summary до 130 символов с ellipsis. Добавлены фильтры по статусу с кнопками all/ready/review/draft. Теги стали clickable chips для фильтрации по конкретному тегу. Реализован toggle "Ready only" для показа только готовых страниц. Breadcrumbs показывают путь Home / Stories / Title для эпизодов.

**Тех-вставка.** Улучшения описаны в PR "chore: explorer UX polish (cards+filters+breadcrumbs)". Карточки обновлены в `createCard()`, фильтры реализованы в `renderIndex()`, breadcrumbs добавлены в `renderPage()`. Состояние фильтров сохраняется в localStorage для восстановления при перезагрузке.

**Что дальше.** После базового UX polish следующий шаг — дальнейшее развитие интерфейса и добавление новых возможностей навигации. Это позволит улучшить пользовательский опыт и сделать Explorer более функциональным.
