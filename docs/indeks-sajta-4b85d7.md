### Как читать метки

- Формат slug: theme/ml, tool/stable-diffusion, role/client
- Синонимы: aliases → canonical, пример: aliases ["машинное обучение", "ML"] → theme/ml
- Видимость: audience:guest, audience:client, audience:editor, audience:dev; internal:* скрыто из UI
- Маппинг: internal:theme/ml → ярлык «Машинное обучение» в фасете «Тема»
- Пример записи:
    - canonical_slug: theme/ml
    - aliases: ["машинное обучение", "ML"]
    - audience: ["guest", "client"]
    - related_to: [tool/stable-diffusion, tool/transformers]
