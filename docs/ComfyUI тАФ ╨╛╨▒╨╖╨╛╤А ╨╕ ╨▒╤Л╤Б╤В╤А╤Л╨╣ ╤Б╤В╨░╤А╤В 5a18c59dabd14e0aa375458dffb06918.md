# ComfyUI — обзор и быстрый старт

Источник 1: https://github.com/comfyanonymous/ComfyUI
Рубрика: Инструменты
Статус: Черновик
Уровень материала: Старт
Экспортные форматы: Видео, Пост

### TL;DR

Что это: конструктор workflow для генерации изображений. Зачем: гибкая сборка пайплайнов без кода. Когда использовать: быстрые эксперименты и воспроизводимые пресеты.

### Быстрый старт

1. Установка по инструкции из репозитория
2. Запустить UI и загрузить базовый workflow
3. Сохранить пресет и экспортировать результат

### Частые ошибки

- Конфликты версий моделей и узлов
- Нехватка VRAM — снижать резолюцию, использовать fp16

### Интеграции

- Экспорт пресетов для публикаций
- Связь с портфолио (кейсы)
