# Think Tank — компактное ядро

## TL;DR
- Компактное ядро — четыре файла, которые дают полную картину проекта за пару минут.
- Notion остаётся авторским источником, GitHub — проверяемое зеркало для публикации и CI.
- Любое изменение проходит через цикл «Обсудили → Разложили → Связали».

## Что входит
- `goals.md` — цели, ценность и критерии успеха.  
- `organization.md` — инфраструктура, стек и процессы поддержки.  
- `agents.md` — реестр AI-агентов и ролей.  
- `focus.md` — текущие приоритеты и ограничения по времени.

## Принципы
- **Source of truth.** Правки делаем в Notion, затем синхронизируемся через `notion-sync`.  
- **Зеркало без сюрпризов.** GitHub показывает только нормализованные данные и историю PR.  
- **Навигация по ролям.** Каждый файл содержит CTA для клиента, продюсера и разработчика.

## Как поддерживать
1. Обновляем разделы в Notion, прогоняем импорт и линты.  
2. Проверяем, что TL;DR актуален и ссылки ведут на свежие материалы.  
3. В журнале ADR фиксируем решения, которые меняют структуру ядра.
