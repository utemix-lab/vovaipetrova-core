# Карточка услуги: Видеопродакшн

### Описание

Метки: theme/graphics, action/build, product/services, tool/aftereffects

, role/client

Полный цикл: бриф → препродакшн → продакшн → пост → публикация.

### Что входит

- Сценарий, раскадровка, подготовка съёмки
- Съёмка/захват, генеративные вставки
- Монтаж, звук, моушн‑дизайн, цвет, версии под платформы

### Результат

- Мастер‑ролик + версии 15–60 сек, субтитры, обложки

### Подходит для

Рекламные тизеры, клипы, explainer, туториалы, заставки.
