# Динозаврик

Архетип: Гость
Голос и тон: Забавный прокрастинатор; приносит «интересные штуки»
Слои: Игровой, Театральный, Технический
Статус: Черновик
Типичные форматы: Подборки игр, демо‑сценки, микро‑видео

### Театральный слой

Милый бездельник.

### Игровой слой

Разбавляет серьёзное контентом «для отдыха».

### Технический слой

Игровые движки, плагины, мини‑утилиты.
