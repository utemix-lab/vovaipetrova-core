
В Obsidian можно добавить иконки перед названиями папок в дереве каталога с помощью плагинов — проще всего это делать через **Icon Folder** (раньше назывался **Iconize**). Этот плагин специально создан для назначения иконок к папкам и файлам прямо в боковой панели.[](https://fedorovpishet.ru/pimp-my-obsidian/)​

## Icon Folder (или Iconize)

Этот плагин позволяет:

- назначать иконки любой папке или файлу через контекстное меню («Change Icon»);
    
- использовать внешние библиотеки вроде **Material Icons**, **Remix Icons** или **Font Awesome**;
    
- раскрашивать иконки и подбирать стиль под тему Obsidian;
    
- хранить индивидуальные назначения иконок для каждой папки.
    

## Установка

1. Открой Obsidian → Настройки → Community Plugins → Включи «Third-party plugins».
    
2. Перейди в вкладку Browse и найди **Icon Folder** (или **Iconize**, если используешь последнюю версию).
    
3. Установи и включи плагин.
    
4. В дереве файлов кликни правой кнопкой по папке → «Change Icon» → выбери иконку из списка.
    

## Дополнительно

Если нужны цветные иконки, можно добавить собственный **icon pack**:

1. Скачай набор иконок в SVG.
    
2. В настройках плагина (`Settings → Iconize → Icon Pack`) создай новый пакет и укажи путь к папке с иконками.[](https://pikabu.ru/story/dlya_yestetov_i_lyubiteley_chut_raskrasit_vneshniy_vid_svoey_bazyi_10923917)​
    

Альтернативный вариант — сделать это через **CSS snippet**, если хочется статичных иконок для конкретных путей, но плагин дает удобный графический интерфейс и не требует знаний CSS.
