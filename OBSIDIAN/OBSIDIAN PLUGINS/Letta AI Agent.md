## Установка

Какие файлы скачивать?

Для ручной установки **нужны три файла** из раздела Assets:

- `main.js`
    
- `manifest.json`
    
- `styles.css`
    

---

## Как установить Letta AI Agent в Obsidian вручную

1. **Скачайте эти три файла** (`main.js`, `manifest.json`, `styles.css`) на свой компьютер.
    
2. **Откройте папку с плагинами Obsidian:**
    
    - В Obsidian перейдите: Настройки → Плагины сообщества → Открыть папку плагинов.
        
    - Откроется папка `.obsidian/plugins/` в вашем хранилище (vault).
        
3. **Создайте там новую папку** по названию плагина, например `letta-obsidian` (английскими буквами без пробелов).
    
4. **Скопируйте внутрь этой папки загруженные файлы** (`main.js`, `manifest.json`, `styles.css`).
    
5. **Перезапустите Obsidian** или отключите/включите плагины сообщества.
    
6. **Активируйте Letta AI Agent** через раздел “Плагины сообщества” → найдите Letta AI Agent и включите переключателем.

## Где взять Letta API Key и Project ID?

Чтобы получить **Letta API Key** и **Project ID** для использования Letta AI Agent в Obsidian, нужно создать аккаунт на официальном сервисе Letta. Вот подробная инструкция:

---

## 1. Зарегистрируйся на платформе Letta

- Перейди на сайт: [https://app.letta.com](https://app.letta.com/)[](https://www.letta.com/)​
    
- Создай бесплатный аккаунт (email/Google/Discord).
    
- После регистрации ты окажешься в личном кабинете Letta Cloud.
    

---

## 2. Получи Letta API Key

- В личном кабинете найди раздел **"Settings"** (Настройки).
    
- В настройках выбери "API Keys" или "Tokens" (обычно находится в левой боковой панели).
    
- Сгенерируй новый API ключ ("Generate new API Key" или "+ Add Key") — он будет выглядеть как длинная строка из букв и цифр.
    
- Скопируй этот ключ — он понадобится для интеграции с Obsidian.
    

Подробно — официальное руководство тут:  
[https://docs.letta.com/guides/cloud/letta-api-key](https://docs.letta.com/guides/cloud/letta-api-key)[](https://docs.letta.com/guides/cloud/letta-api-key)​

---

## 3. Найди или создай Project ID

- На главной панели Letta обычно отображаются "Projects" или “Мои проекты”.
    
- Выбери нужный проект или создай новый.
    
- Project ID отображается в свойствах/настройках проекта (обычно в формате `prj-xxxxxxxx` или похожем).
    
- Если не можешь найти Project ID явно, загляни в адресную строку браузера — после `/project/` будет нужное значение.
    

---

## 4. Введи эти данные в настройках Letta AI Agent в Obsidian

- Открой Obsidian → настройки плагина Letta AI Agent.
    
- Вставь свой API Key и Project ID в соответствующие поля.
    
- Сохрани настройки — теперь твой Obsidian сможет автоматически синхронизировать заметки и историю с ИИ-агентом Letta.
    

---

## Важно!

- Никому не показывай свой API ключ: это твой доступ к аккаунту, и по нему можно управлять твоими агентами.
    
- Если используешь Letta локально (self-hosted), способ получения ключей может отличаться — инструкция обычно есть на GitHub проекта.
