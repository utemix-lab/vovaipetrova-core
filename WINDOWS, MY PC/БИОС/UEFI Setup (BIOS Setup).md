
UEFI Setup (иногда называют просто “UEFI” или “BIOS Setup”) — это графический предзагрузочный интерфейс для настройки всех ключевых параметров материнской платы: порядок загрузки, работа с накопителями, разгон, мониторинг, параметры вентиляторов и др. На ASUS SABERTOOTH Z97 MARK 2 используется современный графический UEFI с поддержкой управления мышкой и подробной структурой меню.[](https://ru.gecid.com/mboard/asus_sabertooth_z97_mark_2/)​​

## Как войти в UEFI Setup

- При запуске компьютера сразу после включения несколько раз нажимай клавишу **Del** (или иногда F2) до появления интерфейса UEFI ASUS.[](https://www.asus.com/ru/support/faq/1049829/)​
    

## Как выглядит и как пользоваться

- **EZ Mode** — простой режим для базовых операций (выбор загрузочного устройства, профиль вентиляторов, время и дата, общая информация).[](https://www.hardwareluxx.ru/index.php/artikel/hardware/mainboards/31030-sabertooth-z97-mark-i.html?start=5)​
    
- **Advanced Mode** (переключение по F7) — расширенный режим для тонкой настройки памяти, CPU, напряжений, управления вентиляторами, разгонных профилей, параметров SATA и, при наличии, NVMe/PCIe/RAID.​[](https://ru.gecid.com/mboard/asus_sabertooth_z97_mark_2/)​
    
- Навигация — мышкой, стрелками, Enter и Tab; работу с настройками сопровождает подробная справка и подсказки в боковой панели меню.
    
- Все изменения подтверждаются нажатием F10 перед выходом (“Save and Exit”).
    

## Где настраивать ключевые параметры

- **Раздел “Boot”** — настройка порядка загрузки, включение/выключение UEFI/Legacy boot, настройка опций загрузки с USB/NVMe/SATA.[](https://www.asus.com/ru/support/faq/1049829/)​
    
- **AI Tweaker** — разгон CPU, настройка памяти, управление профилями питания.[](https://ru.gecid.com/mboard/asus_sabertooth_z97_mark_2/)​
    
- **Monitor/Advanced** — контроль температур, напряжений, выставление параметров системы охлаждения.[](https://www.hardwareluxx.ru/index.php/artikel/hardware/mainboards/31030-sabertooth-z97-mark-i.html?start=5)​
    

## Важно

- Всегда сохраняй изменения F10.
    
- В интерфейсе предусмотрена кнопка сброса “Load Optimized Defaults” — чтобы быстро вернуть заводские настройки.
    

**UEFI Setup — это ключевой инструмент для продвинутых пользователей: в нем настраивают всё железо до загрузки Windows, включая загрузку с NVMe, скорость вентиляторов, работу оперативной памяти и чтение информации о компонентах**.​[](https://ru.gecid.com/mboard/asus_sabertooth_z97_mark_2/)​
