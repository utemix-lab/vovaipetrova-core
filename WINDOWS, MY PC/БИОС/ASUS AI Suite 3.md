
Для старых плат, таких как ASUS SABERTOOTH Z97 MARK 2, отдельную утилиту ASUS EZ Update зачастую можно найти именно в составе фирменного пакета **ASUS AI Suite 3**.

Вот реальные варианты, где искать и скачивать:

1. **GitHub-репозиторий с репаком AI Suite 3**  
    [https://github.com/levelKro/AI-Suite-III-Repack](https://github.com/levelKro/AI-Suite-III-Repack)  
    Там можно скачать обновлённый архив репака, где EZ Update включён в набор. Прямая ссылка в самом репозитории — последняя рабочая версия подходит для большинства старых ASUS. После скачивания выбирай нужные компоненты (EZ Update, System Info и др.).[](https://github.com/levelKro/AI-Suite-III-Repack)​
    
2. **Альтернативные сайты-архивы драйверов**  
    [https://asus.download-windows.org](https://asus.download-windows.org/)  
    Здесь часто бывает утилита ASUS Update Utility/EZ Update для старых плат, иногда неофициальные архивы. Для Windows версий до 10/11.[](https://asus.download-windows.org/)​
    
3. **Если нужен AI Suite 3 оригинал для Z97:**  
    На глобальном сайте ASUS (если модель не находится по прямому поиску), попробуй найти через раздел “Downloads” для других плат серии Z97, так как пакет может быть универсальным для всех чипсетов Z97, включая MARK 2.
    
4. **Прямой способ:**
    
    - Введи в поиске “ASUS AI Suite 3 download zip” и ищи по драйверным архивам форума hwdrivers.com, driver.ru и аналогичных.
        

**Важно!**  
EZ Update — часть пакета AI Suite 3 для Windows, отдельной версией сейчас почти не распространяется.[](https://asus.download-windows.org/)​

Если не удается найти фирменный источник, используй официальный или репак-версию из упомянутого GitHub, проверяя совместимость — это наиболее надежный способ для такой старой платы.

В большинстве случаев различие между **AsusSetup.exe** и **Setup.exe** следующее:

- **AsusSetup.exe** — фирменный установщик от ASUS, автоматизирует установку всех компонентов и утилит. Обычно запускается для полной установки пакета для материнских плат ASUS, включая такую специфическую интеграцию, как AI Suite 3 или драйверы.[](https://otvet.mail.ru/question/210920548)​
    
- **Setup.exe** — стандартный универсальный установщик, может запускать только базовую процедуру установки или быть частью драйверов, не всегда включает дополнительные утилиты ASUS.
    

**Для установки ASUS AI Suite 3 (и входящего EZ Update):**  
**Запускай AsusSetup.exe** — он гарантирует установку всех нужных компонентов, проверяет совместимость и корректно интегрирует утилиты с системой. Это рекомендуемый путь для материнских плат ASUS любого поколения.[](https://www.asus.com/ca-en/support/faq/1012780/)​

Если AsusSetup.exe не запускается или выдаёт ошибку, попробуй Setup.exe, но обычно именно AsusSetup.exe нужен для полной установки AI Suite 3.

Для ASUS SABERTOOTH Z97 MARK 2 максимально полезными и рабочими будут следующие компоненты из списка на скриншоте:

- **Dual Intelligent Processors 5** — основная оболочка AI Suite 3, позволяет управлять питанием, частотами, мониторингом температуры и разгона. Удобно для общей настройки системы даже на старых платах.
    
- **EZ Update** — утилита для простого обновления BIOS и других фирменных компонентов. Самый безопасный способ обновлять BIOS для вашей платы.
    
- **System Information** — показывает подробные сведения о чипсетах, BIOS, устройствах, напряжениях и т.п.
    
- **USB 3.0 Boost** — ускорение работы USB-устройств, если есть сторонние быстрые накопители, может дать небольшой прирост пропускной способности USB-портов (на Z97 работает частично, в основном только на оригинальных ASUS USB-контроллерах).
    

## Опциональные/могут не работать или не нужны:

- **Fan RGB LED Control** — на вашей материнке подсветка не используется, этот компонент малополезен (Z97 MARK 2 не имеет RGB).
    
- **File Transfer** — утилита для ускоренного обмена файлами по USB, полезна крайне редко, особой разницы с обычным копированием вы не заметите.
    

**Рекомендую ставить:**

- Dual Intelligent Processors 5
    
- EZ Update
    
- System Information
    
- USB 3.0 Boost (если есть быстрые внешние диски/флешки, иначе опционально)
    

Остальные компоненты можно смело отключить на этапе установки, чтобы не загружать систему лишними утилитами.
