
Нажмите на ссылку Change adapter options﻿ в разделе Related settings. Это откроет окно Network Connections﻿.

В этом окне:

1. Найдите подключение с названием Ethernet﻿.
    
2. Кликните по нему правой кнопкой мыши и выберите Properties﻿.
    
3. Найдите и выделите строку Internet Protocol Version 4 (TCP/IPv4)﻿, затем нажмите кнопку Properties﻿.
    
4. В открывшемся окне убедитесь, что выбраны опции:
    
    - Obtain an IP address automatically﻿
        
    - Obtain DNS server address automatically﻿
        
5. Нажмите OK﻿, а затем закрывайте окна.
    

Это позволит вашему ПК автоматически получать правильные сетевые настройки и, скорее всего, решит проблему с "Unidentified network".
