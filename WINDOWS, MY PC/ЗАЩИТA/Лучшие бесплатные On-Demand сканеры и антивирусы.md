
- **ClamWin Free Antivirus**
    
    - Открытое ПО, бесплатное, не имеет модуля постоянной защиты.
        
    - Сканирует только по команде (через интерфейс или контекстное меню «Проверить этим антивирусом»), не нагружает систему.[](https://ab57.ru/clamsentinel.html)​
        
    - Позволяет настраивать автоматический запуск, если это нужно, но по умолчанию постоянно не работает.
        
- **Dr.Web CureIt!**
    
    - Бесплатная утилита, которая не устанавливается в систему, запускается вручную для разовых проверок.[](https://vellisa.ru/best-antivirus-scanners)​
        
    - Не конфликтует с другим ПО и идеальна для периодических сканирований.
        
- **Kaspersky Virus Removal Tool**
    
    - Бесплатный сканер, не требует установки и работает только по запуску пользователя.[](https://www.comss.ru/list.php?c=virus_scanners)​
        
    - Подходит для ручных глубоких сканирований.
        
- **ESET Online Scanner**
    
    - Не имеет модуля постоянной защиты — запускается вручную, сканирует выбранные области или всю систему.[](https://www.eset.com/us/home/online-scanner/)​
        
- **Microsoft Safety Scanner (MSERT)**
    
    - Официальный инструмент Microsoft для разовых проверок, не оставляет компонент в системе после завершения работы.[](https://learn.microsoft.com/ru-ru/defender-endpoint/safety-scanner-download)​
        
    - Можно скачать и запустить проверку на вирусы, работает, как отдельное приложение.
        
- **VirusTotal**
    
    - Онлайновый сервис для проверки отдельных файлов по 80+ антивирусным движкам. Хорошо подходит для единичных подозрительных файлов: upload → instant scan.[](https://www.reddit.com/r/antivirus/comments/n6nzpl/looking_for_an_on_demand_only_virus_scanner/)​
        

## Как использовать

- Установите или скачайте нужный сканер.
    
- Запускайте его только при надобности — вручную, либо через контекстное меню.
    
- Можно запускать полную проверку системы или отдельных файлов/папок.
    
- После сканирования программа не вмешивается в работу компьютера до следующего запуска.
    

**Вывод:** такие сканеры отлично подходят для продвинутых пользователей, которым не нужна постоянная защита. Самые удобные: ClamWin, Dr.Web CureIt!, Kaspersky Virus Removal Tool, Microsoft Safety Scanner и онлайн‑сканеры вроде VirusTotal или ESET.[](https://vellisa.ru/best-antivirus-scanners)​
