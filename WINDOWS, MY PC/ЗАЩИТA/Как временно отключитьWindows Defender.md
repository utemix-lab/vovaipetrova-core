
Временно отключить Windows Defender (Microsoft Defender) можно несколькими простыми способами, официально поддерживаемыми самой ОС:[](https://kod.ru/how-to-disable-windows-defender)​

## Через настройки Windows

1. Откройте «Параметры» (Win+I).
    
2. Перейдите в раздел «Обновление и безопасность» → «Безопасность Windows» → «Защита от вирусов и угроз».
    
3. Найдите блок «Параметры защиты от вирусов и угроз» и кликните «Управление настройками».
    
4. Отключите параметр «Защита в реальном времени» — система временно отключит антивирус до следующей проверки или перезапуска, после чего он снова активируется.[](https://pro32.by/by/article/kak-otklyuchit-antivirus-windows/)​
    

## Через командную строку (CMD, от администратора)

- Выполните команду:
```
sc stop WinDefend
```

- — это отключит службу Defender, но система может перезапустить её автоматически либо после перезагрузки.[](https://pro32.by/by/article/kak-otklyuchit-antivirus-windows/)​
    
- Чтобы включить обратно:

```
sc start WinDefend
```

- Отключение через службу — чуть менее безопасный способ, рекомендуется использовать только для опытных пользователей.
    

## Важно

- После отключения защиту лучше включить снова сразу после завершения нужных действий.
    
- Отключение Defender увеличивает риски заражения системы.
    

Оба способа подходят для временной деактивации встроенной защиты.[](https://adminvps.ru/blog/kak-navsegda-otklyuchit-zashhitnik-windows-10/)​
