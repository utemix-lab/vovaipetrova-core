## Как создать GitHub Personal Access Token:

### **Шаг 1: Заходим в настройки GitHub**

1. Открой [github.com](https://github.com/)
2. Нажми на свой аватар (правый верхний угол)
3. Выбери **Settings** (Настройки)

### **Шаг 2: Переходим в Developer settings**

1. В левом меню прокрути вниз
2. Найди **Developer settings** (в самом низу)
3. Нажми на него

### **Шаг 3: Создаём токен**

1. Выбери **Personal access tokens** → **Tokens (classic)**
2. Нажми **Generate new token** → **Generate new token (classic)**
3. Введи пароль GitHub для подтверждения

### **Шаг 4: Настраиваем права (scopes)**

**Для работы со мной выбери:**

- ✅ repo (полный доступ к репозиториям)
- ✅ read:org (если есть организации)
- ✅ user (доступ к профилю)

**Срок действия:**

- Рекомендую **90 days** или **No expiration** (бессрочный)

### **Шаг 5: Генерируем**

1. Нажми **Generate token**
2. **⚠️ ВАЖНО:** Скопируй токен СРАЗУ! Он показывается только один раз
3. Сохрани в безопасное место

---

**Прямая ссылка для быстрого доступа:** [https://github.com/settings/tokens/new](https://github.com/settings/tokens/new)
