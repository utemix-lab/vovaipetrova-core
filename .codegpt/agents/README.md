# CodeGPT Agents — Промпты и конфигурация

Эта директория содержит промпты и конфигурацию для специализированных CodeGPT агентов.

## Структура

```
.codegpt/agents/
├── README.md              # Этот файл
├── orchestrator.md        # Промпт для CodeGPT:Orchestrator
├── docs.md                # Промпт для CodeGPT:Docs
├── refactor.md            # Промпт для CodeGPT:Refactor
└── creative.md            # Промпт для CodeGPT:Creative
```

## Типы агентов

### CodeGPT:Orchestrator

**Назначение**: Координация задач, планирование, приоритизация.

**Используется для**:
- Планирование спринтов
- Приоритизация задач
- Координация зависимостей
- Анализ метрик

### CodeGPT:Docs

**Назначение**: Работа с документацией.

**Используется для**:
- Создание документации
- Обновление документации
- Рефакторинг структуры docs
- Проверка качества документации

### CodeGPT:Refactor

**Назначение**: Рефакторинг кода.

**Используется для**:
- Улучшение структуры кода
- Оптимизация производительности
- Исправление технического долга
- Улучшение читаемости

### CodeGPT:Creative

**Назначение**: Творческие задачи.

**Используется для**:
- Создание Stories
- Разработка персонажей
- Создание контента
- UX решения

## Использование

1. Скопируйте соответствующий промпт в настройки CodeGPT
2. Настройте контекст проекта (`.codegpt/context.md`)
3. Создайте задачу в Briefs с соответствующим Executor
4. Агент выполнит задачу согласно промпту

## Обновление промптов

Промпты должны регулярно обновляться на основе:
- Обратной связи от команды
- Результатов работы агентов
- Изменений в процессе разработки
- Новых требований проекта
