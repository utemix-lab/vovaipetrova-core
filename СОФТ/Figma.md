
Для дизайнеров существует великое множество программ, и каждая из них по-своему уникальна. Одним из самых популярных продуктов для [[UX_UI-дизайн]] является Figma.

**Figma** — **онлайн-сервис для дизайнеров, веб-разработчиков и маркетологов**. Он предназначен для создания прототипов сайтов или приложений, иллюстраций и векторной графики. [2](https://timeweb.com/ru/community/articles/chto-takoe-figma-i-kak-ey-polzovatsya)

**Некоторые особенности Figma:**

- **Облачное хранение файлов**. Нет необходимости скачивать проект каждый раз после внесения изменений — всё сохраняется на диске автоматически. [4](https://skillbox.ru/media/design/chto_takoe_figma/)[2](https://timeweb.com/ru/community/articles/chto-takoe-figma-i-kak-ey-polzovatsya)
- **Фреймы**. Это отдельное полотно, внутри которого дизайнеры проектируют экраны приложений. Разработчики предусмотрели несколько стандартных размеров под разные устройства: все виды iPhone и самые популярные смартфоны с Android. [4](https://skillbox.ru/media/design/chto_takoe_figma/)
- **Компоненты**. С их помощью можно задать общий стиль и в случае необходимости быстро изменить его. [4](https://skillbox.ru/media/design/chto_takoe_figma/)
- **Многопользовательский режим редактирования**. Есть возможность создания собственной команды для дальнейшего сотрудничества в рамках сервиса. [4](https://skillbox.ru/media/design/chto_takoe_figma/)[2](https://timeweb.com/ru/community/articles/chto-takoe-figma-i-kak-ey-polzovatsya)

Figma работает на Windows, Mac, Linux. Её можно установить на компьютер или работать через браузер. Есть приложение на смартфон — и для iOS, и для Android. [3](https://practicum.yandex.ru/blog/chto-takoe-figma-dlya-dizainera/)
