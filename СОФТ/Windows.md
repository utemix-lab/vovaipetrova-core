
## Утилиты:

- SageThumbs - Windows не поддерживает отображение иконок PSD в проводнике. Это связано с тем, что файлы PSD предназначены в первую очередь для программы Photoshop, а Windows не имеет встроенной утилиты для их открытия.  Для просмотра иконок PSD можно использовать сторонние программы. 
  **SageThumbs**. Бесплатная утилита добавляет поддержку кодеков для сотен типов файлов в проводник Windows. После установки программы нужно открыть окно проводника и перейти в режим просмотра иконок.
https://sagethumbs.com/ - не скачивается
https://sourceforge.net/projects/sagethumbs/

- Office Tab Enterprise 14.50 - вкладки в Microsoft Office (Word, Excel, PowerPoint, Publisher, Access, Project и Visio)
https://rutracker.org/forum/viewtopic.php?t=6523819

- Clover · OvArt - вкладки в Проводнике · Windows 10 стиль
