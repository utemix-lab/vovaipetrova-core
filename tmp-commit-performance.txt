perf: incremental builds with mtime cache

- Добавлена проверка mtime для docs/* файлов
- Реализовано кеширование в prototype/data/.build-cache.json
- Пересборка только изменённых файлов
- Обновление HTML страниц только для изменённых файлов
- Добавлен флаг --force для полной пересборки
- Кеш добавлен в .gitignore

Производительность:
- Полная сборка: ~598ms
- Инкрементальная сборка (все файлы в кеше): ~485ms
- При изменении 1 файла: обрабатывается только изменённый файл
