**Flow State в [[Leonardo.ai]]** — это функция для ускорения процесса генерации изображений и облегчения поиска оптимальных визуальных решений. Она позволяет пользователям быстро создавать большое количество изображений по одному запросу, а затем уточнять результаты, генерируя вариации на основе выбранных стилей.

**Как работает Flow State**:

1. **Инициация**. В разделе создания изображений нужно выбрать режим Flow State.
2. **Ввод запроса**. Можно ввести текстовый запрос или использовать предложенные примеры.
3. **Настройки стиля**. На боковой панели можно выбрать до четырёх предустановок стиля: настроение (Vibe), освещение (Lighting), тип кадра (Shot Type) и цветовая тема (Color Theme). По умолчанию эти параметры рандомизируются, но их можно настроить вручную или отключить.
4. **Генерация изображений**. Нужно нажать «Generate» для начала процесса. Flow State будет непрерывно создавать изображения по мере прокрутки страницы.
5. **Уточнение результатов**. Если какое-то изображение понравилось, нужно нажать «More Like This», чтобы создать поток изображений с похожим стилем.
6. **Сохранение и улучшение**. Понравившиеся изображения можно сохранить или улучшить с помощью функции Upscale для повышения качества.

Flow State доступен всем пользователям Leonardo AI, хотя для бесплатных аккаунтов предусмотрены дневные лимиты на использование.
