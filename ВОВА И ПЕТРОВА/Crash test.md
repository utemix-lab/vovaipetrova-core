Текст для Ютуба:

Вова и Петрова занялись дизайном своих персонажей и дальнейшим развитием собственного бренда. Они тестируют видео-генераторы, а заодно и свои возможные отношения.

Цели ролика:

- Изучить возможности создания последовательного персонажа. Использован метод генерации фотосессий наших героев с помощью функции "Character reference" в Leonardo.ai и последующая загрузка изображений в надстройку Train / LoRA на базе Krea.ai. Этот маневр позволяет получить, если не больший контроль над внешностью персонажа, то, по крайней мере, большую вариативность генераций.
- Протестировать видео-генераторы. Из множества перепробованных, для этой работы подошли Runway и Luma.
- Создать сюжет с наибольшим охватом виртуального измерения, учитывая пока еще весьма ограниченные возможности нейросетей.
- Продемонстрировать возможные варианты оформления видеоконтента (отбивки, заставки, параллельные мини-сюжеты).
