
## Локальные модели и UI для генерации изображений и видео

- **Stable Diffusion** — базовая и наиболее популярная модель для генерации изображений, работает локально на Linux.
    
- **Comfy UI** — современный, модульный интерфейс для работы со Stable Diffusion и производными, очень гибкий, полностью кроссплатформенный.
    
- **Automatic 1111** — популярный веб-интерфейс для Stable Diffusion, поддерживает плагины и расширения, можно ставить на Linux.
    
- **Invoke AI** — ещё одна удобная оболочка для Stable Diffusion с локальным запуском.
    
- **Mochi 1** и **LTX Video** — локальные видеогенераторы AI, работающие на Linux с GPU NVIDIA.
    
- **Stable Video Diffusion (SVD)** — модель для видео, запускается локально через Linux.
    
- **Deforum** — плагин для создание анимаций на основе Stable Diffusion.
    
- **RunwayML** — есть Linux-версия, интегрирует несколько моделей генерации видео и изображений.
    
- **Dreambooth** — для тонкой донастройки моделей на своих данных.

## Модели и среды для NLP и ИИ-проектов

- **LLaMA** и её форки (LLaMA 2, Alpaca, Vicuna) — модели от Meta для текстового ИИ, хорошо запускаются на Linux.
    
- **Hugging Face Transformers** — библиотека, поддерживающая тысячи моделей и удобная для локальной разработки.
    
- **GPT-Neo / GPT-J / GPT4All** — открытые языковые модели, работают на Linux.
    
- **LangChain** — framework для разработки ИИ агентов и приложений на Python.

## Совместимость и установка на Linux

- Все перечисленные модели и UIs поддерживают Linux, чаще всего базуются на Python и PyTorch, с поддержкой GPU NVIDIA (CUDA).
    
- Надежные дистрибутивы: Ubuntu, Debian, Pop!_OS.
    
- Требуется установить драйвера NVIDIA, CUDA, cuDNN.
    
- Модель можно запускать в виртуальных окружениях (conda, venv) или docker.
    
- Управлять видео и изображениями можно через Jupyter, терминал или веб-интерфейсы.
