Добавлена интеграция CodeGPT для работы с Notion и GitHub через их API.

## Что добавлено

- Скрипты для работы с GitHub API (создание PR, получение issues)
- Скрипты для работы с Notion API (поиск, получение, обновление страниц)
- Документация по настройке CodeGPT (docs/codegpt-setup.md)
- Обновлен .gitignore для защиты конфиденциальных файлов
- Обновлен README.md с разделом про CodeGPT интеграцию
- Добавлены npm-скрипты для быстрого доступа

## Безопасность

Конфиденциальные файлы (.env, codegpt.config.json, vscode-settings.example.json) остаются только локально и не попадают в репозиторий благодаря .gitignore.
