
## Чтобы подключить Mac к ПК через роутер

Когда все физические подключения сделаны (кабели Ethernet или Wi-Fi), нужно настроить сеть и доступ на обеих машинах. Вот пошаговая инструкция:

## 1. Проверить сетевые настройки на ПК и Mac

- Убедитесь, что ПК и Mac подключены к одному роутеру и находятся в одной подсети (обычно роутер назначает IP автоматически через DHCP).
    
- На ПК:
    
    - Запустите команду `ipconfig` (Windows) в командной строке и проверьте IP-адрес (например, 192.168.1.X).
        
- На Mac:
    
    - Откройте «Системные настройки» → «Сеть» и проверьте IP-адрес в разделе Ethernet или Wi-Fi.
        

## 2. Настроить общий доступ к файлам

Если хотите обмениваться файлами между ПК и Mac:

## На ПК (Windows 10/11):

- Откройте «Панель управления» → «Сеть и Интернет» → «Центр управления сетями и общим доступом».
    
- Включите сетевое обнаружение и файловый доступ.
    
- Настройте общий доступ к нужным папкам с правами доступа для нужных пользователей.
    
- Убедитесь, что обе машины в одной рабочей группе (Workgroup), чтобы проще находились друг друга.
    

## На Mac:

- Откройте «Системные настройки» → «Общий доступ».
    
- Включите «Общий доступ к файлам».
    
- При необходимости добавьте папки для общего доступа и выберите, кто может их читать/писать.
    
- Включите «SMB» (Windows File Sharing) в разделе «Опции...».
    

## 3. Подключение с Mac к Windows

- В Finder нажмите `Cmd+K` или в меню "Перейти" → «Подключиться к серверу».
    
- Введите адрес Windows ПК, например:

```
smb://192.168.1.X
```

- Введите имя пользователя и пароль Windows для доступа.
    

## 4. Подключение с Windows к Mac

- В Windows проводнике в адресной строке введите:

```
\\IP_адрес_Mac
```

- Введите учетные данные Mac, если потребуется.
    

## 5. Проверить и настроить брандмауэр и антивирус

- Убедитесь, что на ПК и Mac не блокируется трафик для файлового обмена.
    
- Разрешите по нужным протоколам (SMB для Windows, AFP/SMB для Mac) соответствующие подключения.
    

## 6. Дополнительное (по желанию)

- Можно настроить общий доступ к принтеру, медиаконтенту и др. через протоколы DLNA, AirPlay, или через серверы Samba/NFS.
    
- Для удаленного управления можно включить удаленный рабочий стол/RDP на ПК и VNC/Screen Sharing на Mac.

## Работа с дисками через Командную строку (Diskpart)

1. **Запустите Командную строку от имени Администратора.**
    
    - Нажмите `Win + S`, введите `cmd`.
        
    - В правой части окна выберите "Запуск от имени администратора".
        
    - Подтвердите запрос контроля учетных записей (UAC).
        
2. **Запустите утилиту `diskpart`.**  
    Введите команду:

```
diskpart
```

3. **Выведите список всех дисков.**  
Введите команду:

```
list disk
```

Вы увидите список всех физических дисков в системе. **Крайне важно определить номер именно Диска 3** (ориентируйтесь по размеру, ~931 GB). Предположим, это `Диск 2` (так как нумерация в diskpart часто начинается с 0).

_Пример:_

Диск 0 Размер: 3726 GB
Диск 1 Размер: 931 GB
Диск 2 Размер: 931 GB  <-- Это скорее всего наш кандидат

4. **Выберите нужный диск.**  
Будьте предельно внимательны! Если выберете не тот диск, вы уничтожите все данные на нем.  
Введите команду (замените `X` на реальный номер вашего диска из предыдущего шага, например, `2`):

```
select disk X
```

5. **Очистите диск от всех разделов.**  
Эта команда полностью сотрет всю структуру разделов и данные на выбранном диске.

```
clean
```

Утилита ответит: `DiskPart: очистка диска выполнена успешно`.

6. **Создайте новый основной раздел на всем пространстве диска.**

```
create partition primary
```

Ответ: `DiskPart: указанный раздел создан`.

7. **Отформатируйте созданный раздел в файловую систему NTFS.**  
    Вы можете задать метку тома (например, `DATA`) или оставить ее пустой. Процесс форматирования займет некоторое время.

```
format fs=ntfs quick label="VIP"
```

 - `fs=ntfs` — указываем файловую систему.
        
    - `quick` — быстрое форматирование (достаточно, если диск новый или данные не нужны).
        
    - `label="VIP"` — задает имя диска (метку тома). Можете назвать его как угодно.
        
8. **Назначьте букву диска.**  
    Система обычно делает это автоматически, но можно сделать и вручную:

```
assign
```

 Ответ: `DiskPart: назначение имени диска или точки подключения выполнено успешно`.

9. **Выйдите из diskpart.**

```
exit
```
