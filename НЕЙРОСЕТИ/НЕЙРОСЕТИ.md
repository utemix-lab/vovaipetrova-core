**Код стиля для генерации изображений** — это **уникальный код, который позволяет задать определённый стиль или эстетику при создании картинки**. [3](https://vc.ru/midjourney/1456975-10-luchshih-sref-kodov-stilei-dlya-midjourney-dlya-potryasayushih-illyustracii-kotorye-vam-ponravyatsya)







Например, в нейросети Midjourney для этого используется функция **Style Tuner**. Чтобы создать свой стиль обработки, нужно набрать команду «/tune» и затем ввести текстовый запрос. После завершения работы создаётся специальная веб-страница уникального стиля, и её адрес отправляется пользователю в личном сообщении. На странице создания стиля можно выбрать понравившийся стиль, чтобы на его основе создать свой собственный персонализированный стиль. Для каждого стиля на странице есть код стиля, который можно скопировать. [1](https://postium.ru/style-tuner-midjourney/)







Также в Midjourney есть **Sref-коды** — это уникальные числа, которые добавляют в промпт. При их использовании Midjourney применяет определённый стиль или эстетику к изображению. [3](https://vc.ru/midjourney/1456975-10-luchshih-sref-kodov-stilei-dlya-midjourney-dlya-potryasayushih-illyustracii-kotorye-vam-ponravyatsya)
