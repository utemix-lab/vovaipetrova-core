
## Цель
Обеспечить стабильное подключение к серверу через SSH без "отлипания" терминала и с сохранением привычного поведения ПКМ-вставки.

## Шаги

### 1. Проверка подключения по SSH
Убедитесь, что вы можете подключиться к серверу вручную:

```bash
ssh root@87.228.88.243
```

Если вход успешен, продолжайте настройку.

### 2. Изменение файла `~/.bashrc` на сервере
Подключитесь к серверу и выполните:

```bash
nano ~/.bashrc
```

Добавьте или раскомментируйте в самом конце:

```bash
exec bash
```

Сохраните файл и примените изменения:

```bash
source ~/.bashrc
```

## Результат
Теперь при подключении к серверу с помощью Git Bash:
- Окно не "отлипает" (не закрывается сразу).
- Подключение работает стабильно.
- Вставка через ПКМ работает штатно (если не отключена в настройках Windows).

## Примечание
Файл `~/.bashrc` загружается при интерактивных входах в систему. Команда `exec bash` гарантирует сохранение оболочки при входе через SSH.
