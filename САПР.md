
**САПР** (система автоматизированного проектирования) — **программное обеспечение для создания, редактирования, анализа и оптимизации проектов**. [secrets.tbank.ru](https://secrets.tbank.ru/glossarij/sapr/)

САПР используют инженеры, дизайнеры и архитекторы для работы с чертежами, схемами и 3D-моделями. Система облегчает работу специалистов, повышает точность и скорость проектирования. [secrets.tbank.ru](https://secrets.tbank.ru/glossarij/sapr/)

**Некоторые виды САПР:**

- **По области применения**:
    - **Универсальные** — подходят для решения разных задач проектирования: измерения и построения объектов, учёта свойств материалов. Пример таких САПР — AutoCAD и SOLIDWORKS.
    - **Специализированные** — подходят для конкретной области. Например, архитектурные — Revit, Archicad; машиностроительные — CATIA; электротехнические — EPLAN, OrCAD; строительные — Tekla Structures.
- **По функциональному назначению**:
    - **Графические** — для создания чертежей и визуализаций. Пример — AutoCAD, CorelCAD.
    - **Инженерные** — для выполнения расчётов и симуляций. Пример — ANSYS, Abaqus.
    - **Для трёхмерного моделирования** — позволяют создавать визуальные 3D-модели. Пример — SOLIDWORKS, Blender.
    - **Для управления данными** — нужны для создания проектной документации. Пример — Teamcenter.
- **По уровню автоматизации**:
    - **Ручные** — пользователь создаёт проекты самостоятельно, автоматизации процесса проектирования почти нет.
    - **Полуавтоматизированные** — программа выполняет некоторые этапы работы, например расчёты или построение однотипных объектов.
    - **Автоматизированные** — программы выполняют весь или почти весь цикл проектирования, от создания модели до анализа и подготовки документации.
